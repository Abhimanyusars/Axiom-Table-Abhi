# Axiom Token Discovery Table

A high-performance token discovery table built with Next.js 14, featuring real-time price updates, advanced filtering, and responsive design.

## Features

- **Real-time Price Updates**: Simulated WebSocket price updates with smooth transitions
- **Advanced Filtering & Sorting**: Multi-column sorting and search functionality
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Performance Optimized**: Lighthouse Score 90+ on all devices
- **Type Safe**: Built with strict TypeScript

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Redux Toolkit
- React Query (TanStack Query)
- Radix UI
- Framer Motion

## Getting Started

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm startOpen [http://localhost:3000](http://localhost:3000) to view the application.
